# Stack Teknologi dalam project ini:

# Backend:

# - PHP 8.4

# - Laravel 11 (MVC Framework)

# - MySQL(database — simpel untuk development)

# Frontend:

# - Tailwind 3.4.13 (CSS framework)

# - Flowbite 3.1.1 – komponen UI siap pakai berbasis Tailwind

# - DataTables 1.13 – tabel interaktif (sorting, searching, pagination).

# - jQuery 3.7 – dependency dan helper untuk DataTables.

# - SweetAlert2 (CDN) – popup notifikasi dan dialog konfirmasi yang interaktif.

# - Vite 6 + laravel-vite-plugin – bundler dan dev server untuk asset frontend.

# Development Tools:

# - XAMPP local server stack (Apache + PHP + SQLite/MySQL) untuk menjalankan aplikasi secara lokal.

# - Composer (PHP package manager)

# - PestPHP 3 – framework testing yang dipakai untuk unit/feature test pada aplikasi ini.

# - Laravel Pint (code formatter)

# - Git (version control)

# Cara cek versi yang terpasang:

# - php -v # versi PHP

# - php artisan --version → versi Laravel

# - composer --version → versi Composer

# - node -v # versi Node.js

# - git --version → versi Git
